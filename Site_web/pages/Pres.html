Presentation of the project

Purpose of the project

Read the colors is a project realized by Maxime DUCOIN within the framework of his engineering school course and more particularly of the minor "Métiers de la création numérique".
Within the framework of this minor, the students were asked to realize, individually or in groups, a project mixing technologies, and more precisely digital domain, with an artistic dimension.


Starting idea

The main idea of the project is to provide several tools to color code a text, I.e. to replace each letter by the color that has been assigned to it and then be able to visualize this text in an innovative way.


Realization

Then this initial idea had to be developed to add more technicality to the project.
Indeed, the following points have been added:

  • Creation of a website

  • Addition of an interactive part

  • Addition of several color palettes


Creation of a website

A website seems to be the best tool to realize this project, indeed, it is easy to use, accessible to all and requires no download.

Addition of an interactive part

Add a more playful aspect in order to extend the attractiveness of this project is necessary, indeed, the basic idea is purely artistic because it makes it possible to visualize a text in color and to give a new dimension to the writings but to make it possible to the people to discover or to learn to read the colors is also interesting.
This new idea would allow users to discover the letters composing the text as they go along. By first seeing the numerical work drawn from the text to then reveal the letters one by one with their cursor.

Addition of several color palettes

In the development of the subject it also seemed necessary to offer the possibility of using different color palettes so that the theme of the digital work is as close as possible to the theme of the text.
